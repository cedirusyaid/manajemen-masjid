Agenda Acara Rundown Dinamis

Rundown Acara & Penjadwalan Dinamis

  • Rundown Terikat Kegiatan: Setiap sub-kegiatan memiliki lini masa (rundown) jam ke jam yang teratur.
  • Relasi Master Personel: Menghubungkan pembicara/Ustadz langsung ke data database masjid untuk menampilkan profil biografi singkat.
  • Countdown Papan Agenda: Halaman utama jemaah menampilkan hitung mundur waktu kajian atau kegiatan terdekat secara dinamis.

Auto-Deployment Deploy.php Webhook

Sistem Auto-Deployment & Pemeliharaan

  • Debian 12 Server Ready: Siap dipasang pada infrastruktur server Diskominfo Kabupaten Sinjai.
  • Webhook Mandiri (`Deploy.php`): Memungkinkan pembaruan website langsung dari GitHub hanya dalam hitungan detik.
  • Bebas Hambatan CSRF: Berkas deploy diletakkan secara terpisah di luar filter CSRF global untuk menghindari error pemblokiran.
  • Keamanan Token: Dilengkapi kunci keamanan dinamis (secret token) agar server terhindar dari trigger liar pihak luar.
